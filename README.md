
### 爱多聊Ai

> 使用Maui blazor hybrid 开发混合多平台应用，支持以下功能：

1. 与ChatGPT聊天（支持域配置官方或代理的）
2. 语音对话（文本转语音-azure服务，语音转文本-建议使用讯飞输入法 注意保护个人隐私通讯录权限谨慎授权）
3. 支持Dall-e 2/3 生成图片
4. 支持会话管理，新增，保存，历史
5. 内容支持markdown格式显示
6. 随时修改会话提示prompt
7. 内置部分练手的prompt项目和awesome prompts项目的近200个提示词供参考使用
8. 支持自定义Prompt
9. 仅必需的权限授权：存储，麦克风，网络
